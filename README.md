# Lightbox
Lightbox добавляет на сайт удобный просмотр изображений на весь экран: клик по картинке открывает галерею с увеличением и навигацией. Отлично подходит для постов, гайдов и виджетов, где изображения нужно рассматривать детально.  **Возможности:** - **Автоподключение** к изображениям внутри `.md-content` (посты и виджеты). - **Галерея внутри блока**:
