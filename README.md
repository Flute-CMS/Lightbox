# Lightbox

Lightbox добавляет на сайт удобный просмотр изображений на весь экран: клик по картинке открывает галерею с увеличением и навигацией. Отлично подходит для постов, гайдов и виджетов, где изображения нужно рассматривать детально.

**Возможности:**
- **Автоподключение** к изображениям внутри `.md-content` (посты и виджеты).
- **Галерея внутри блока**: перелистывание назад/вперёд + счётчик.
- **Зум** колёсиком, клавишами `+/-/0` и двойным кликом.
- **Панорамирование** при увеличении (drag мышью/тачем).
- **Клавиатурное управление**: `Esc` закрыть, `←/→` навигация.
- **Не мешает ссылкам**: картинки, обёрнутые в `<a>`, пропускаются.

---

`Lightbox` adds a clean full-screen image viewer: click any image to open a zoomable gallery with navigation. Great for posts, guides, and widgets where images should be inspected in detail.

**Features:**
- **Auto-attaches** to images inside `.md-content` (posts and widgets).
- **Per-block gallery** with prev/next controls and a counter.
- **Zoom controls** via mouse wheel, `+/-/0` keys, and double click.
- **Pan when zoomed** (mouse/touch drag).
- **Keyboard control**: `Esc` to close, `←/→` to navigate.
- **Link-safe**: skips images already wrapped in anchors.

## Installation

Download the latest release and install it via the Flute CMS admin panel.
